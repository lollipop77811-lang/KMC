# KMC Teal — Design Colors

## CSS Custom Properties (Tailwind `@theme` + `:root`)

| Variable | Hex | Swatch | Usage |
|---|---|---|---|
| `--color-teal` | `#0F766E` | Primary accent — links, buttons, icons, decorative lines, active filters, scrollbar |
| `--color-teal-light` | `#14B8A6` | Footer college label, map/mail icons, nav hover |
| `--color-teal-dark` | `#0B4F4A` | Button hover, scrollbar hover, dark section backgrounds |
| `--color-navy` | `#0F2942` | Headings, nav links, footer bg, card headings, button text |
| `--color-navy-light` | `#1E3A5F` | Button hover states (defined in `@theme` only, used as hardcoded value) |
| `--color-coral` | `#FF6B5E` | Emergency CTA cards, emergency phone button |
| `--color-amber` | `#F5A623` | Department page decorative line, badge, step numbers, star ratings, CTA hover |
| `--color-ivory` | `#FBFAF7` | Page/section background |
| `--color-mist` | `#EEF4F3` | Section background, icon containers, scrollbar track, tag bg |
| `--color-sage` | `#DCE9E6` | Defined but currently unused |
| `--color-text` | `#14241E` | Body text color |
| `--color-text-muted` | `#3D524A` | Subtle body text, descriptions, dates |
| `--color-border` | `#E2EBE8` | Card borders, section dividers, filter borders |
